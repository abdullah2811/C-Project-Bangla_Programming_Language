=== BanglaScript: Bengali-Inspired C Transpiler ===

Description:
A compiler that converts Bangla-like syntax (written in Roman script) 
to standard C code.

Features:
- Supports 50+ Bengali keywords (e.g., jodi=if, lekho=printf)
- Single-file compiler (translator.exe)
- Works with any C compiler (GCC, Clang)

Quick Start:
1. Download translator.exe and run_translator.bat
2. Create a .bpl file (e.g., hello.bpl)
3. Drag file onto run_translator.bat

See GETTING_STARTED.txt for detailed instructions.
